Silpa Source Code

Download the nightly code snapshot in tar.gz format

The latest stable release is available here in tar.gz format.

Silpa uses git for version control. This site is version controlled by git. To get a copy of the source code of silpa, clone the git repository

git clone git://git.savannah.nongnu.org/silpa.git

If you don't have git installed in your machine, install it using the package manager of your operating system. If you are new to git, start from a git tutorial

Browse the git repository using gitweb
